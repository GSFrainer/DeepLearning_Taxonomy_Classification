# Feature-Classifier - Baseline experiments
This folder contains the code of baseline experiments using the q2-feature-classifier plug-in from QIIME2.

Files:
- **analysis.ipynb**: Notebook to generate aggregated experiment metrics
- **qiime_script.sh**: Bash script for batch experiment workflow using QIIME 2
