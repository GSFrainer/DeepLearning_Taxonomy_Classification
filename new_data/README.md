# New Data folder
This folder receives the train-test data splits generated by the data generator script from ``./data`` folder.

> In this repo, there is a sample of data split preloaded with the ``prop_0-2/min_10/RandomSplit_0`` configuration. For the complete data splits used in the experiments, follow the instrucions in [How to Run the Experiments / Reproducibility](https://github.com/GSFrainer/DeepLearning_Taxonomy_Classification/tree/main?tab=readme-ov-file#how-to-run-the-experiments--reproducibility) section of project's main README.



The structure of the folders added by the script varies depending on the date split configuration used.

Ex.:
>  prop_0-2/min_10/RandomSplit_0/species
- prop_0-2 : Train-test ratio as 0.2
- min_10 : Class size threshold - minimum size 10 entries
- RandomSplit : Simple random sampler
- 0 : Seed for reproducibility as 0
- species : Species taxonomy rank
